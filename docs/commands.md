# Команды git

## Сокращения и обозначени

```bash
[] - опциональный аргумент 
<name> - название сущностей(ветка, файл, путь и т.д.)
```

### Инициализация репозитория

``` bash
mkdir <repo>
cd <repo>
git init
```

### Клонирование 

```bash
git clone <url> [dir]
```

### Коммит

1. Добавить изменения в контекст 

```bash
git add <files>
```
2. Сдлеать коммит

Интерактивно 

```bash
git commit
```
С сообщением

```bash
git commit -m <message>
```
